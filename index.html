Database Tool w Administrator Menu

  • Java program designed to allow users to make personal databases and for administrators to remove potentailly troubling information.
  • Program implements serialization to allow user information to only be accessed by the user and the admins. Information is stored using Java file I/O.
  • Abstract Data types allow for common methods of an account to be shared accross both an admin class and a user class.
  • Java file I/O also stores information such as usernames, password, database names, and the contents of each database.
